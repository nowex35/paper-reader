UI言語

アプリの表示言語を切り替えます。解説の言語は「母国語」設定で別途変更できます。

母国語

母国語を選択すると、その言語で解説が生成されます。論文と同じ言語の場合、翻訳セクションは省略されます。

原文が読める場合、解説から翻訳セクションを省き、用語解説と要約のみ表示します。

質問機能

クラウドLLMのAPIキーを設定すると、論文全文を踏まえた質問ができるようになります。翻訳・解説は設定なしで使えます。

APIキーの取得方法

Google Gemini

  1. Google AI Studio を開く
  2. 「APIキーを作成」→ コピーして上の欄に貼り付け → 保存

無料枠で十分使えます。

OpenAI

  1. OpenAI Platform を開く
  2. 「Create new secret key」→ コピーして上の欄に貼り付け → 保存

Anthropic (Claude)

  1. Anthropic Console を開く
  2. 「Create Key」→ コピーして上の欄に貼り付け → 保存

カスタム (OpenAI互換)

LM Studio, Ollama (OpenAI互換モード), vLLM, LocalAI など、OpenAI互換の /v1/chat/completions を提供するサーバーに接続します。

  • ベースURLにサーバーのアドレスを入力（例: http://localhost:1234）
  • APIキーは不要な場合は空欄のままでOK
  • モデル名はサーバーが返す一覧から選択、またはカスタム入力

キーはこの Mac 内にのみ保存され、選択したプロバイダの API への通信にのみ使用されます。
