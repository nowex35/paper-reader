質問機能

クラウドLLMのAPIキーを設定すると、論文全文を踏まえた質問ができるようになります。翻訳・解説は設定なしで使えます。

APIキーの取得方法

Google Gemini

  1. Google AI Studio を開く
  2. 「APIキーを作成」→ コピーして上の欄に貼り付け → 保存

無料枠で十分使えます。

OpenAI

  1. OpenAI Platform を開く
  2. 「Create new secret key」→ コピーして上の欄に貼り付け → 保存

Anthropic (Claude)

  1. Anthropic Console を開く
  2. 「Create Key」→ コピーして上の欄に貼り付け → 保存

キーはこの Mac 内にのみ保存され、選択したプロバイダの API への通信にのみ使用されます。
